# Array Operations Program (C++)

A C++ project that performs operations on arrays.

## Features
- Enter 5 numbers
- Find largest number
- Find smallest number
- Calculate sum
- Calculate average

## Concepts Used
- C++
- Arrays
- Loops
- Conditions
- Variables

## Example Output

Enter 5 numbers:

10
20
30
40
50

Largest number = 50
Smallest number = 10
Sum = 150
Average = 30
